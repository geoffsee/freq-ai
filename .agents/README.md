# .agents - Agent Skills for Freq Cloud

This directory contains [Agent Skills](https://agentskills.io) for AI agents working on the Freq Cloud project. Skills are automatically discovered by compatible agents (Claude Code, Cursor, VS Code Copilot, Gemini CLI, and others).

## Available Skills

| Skill | Description |
| :--- | :--- |
| `freq-cloud-context` | Core project context, priorities, and key resources |
| `freq-cloud-architecture` | Node-based architecture overview and system design |
| `freq-cloud-coding-standards` | Rust, deno_core, and TypeScript coding patterns |
| `freq-cloud-testing` | Test commands, verification workflow, and submission checklist |
| `code-explorer` | Use toak CLI for codebase snapshots and semantic search |
